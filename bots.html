Bots

Automated tools I have built. Uses python run on AWS Lambda. All code and graphics created by myself and partners.

Maryland Federal Register Bot

The Federal Register is a government publication that announces government agency rules, proposed rules, and public notices. The Maryland Federal Register Bot sorts hundreds of entries everyday and tweets any that are relevant to the State of Maryland. Topics normally include Chesapeake Bay conservation, Maryland special events, and local government agency actions.

Maryland Case Search Bot

A customizable Slack bot build to scrape the Maryland Case Search database for interesting cases and post slack messages when new ones are found. I built the current implementation to assist the crime reporter at the Capital Gazette.

Trump Emoji Bot

Twitter bot that translated all of Trumps tweets into emojis and tweets them. Still under construction.
